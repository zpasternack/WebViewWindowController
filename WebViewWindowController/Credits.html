This is a test application for demonstrating WebViewWindowController.

WebViewWindowController is an NSWindowController subclass which includes a WebView. This is an easy way to put content in a window, say, for an About box. It catches clicks on hyperlinks and opens them in the default browser externally. It also observes appearance changes in Mojave, and adjusts the text and hyperlink colors to adapt to dark mode.

WebViewWindowController is written in Swift 5, and was crafted by Zacharias Pasternack for Fat Apps, LLC.
